## DeepSeek（深度求索）⭐ 重点关注

> 2025 年初震惊全球，性能比肩顶尖、成本极低的国产之光

| 模型 | 特点 |
|------|------|
| **DeepSeek V3** | 综合能力极强，代码/数学顶尖，API 极便宜 |
| **DeepSeek R1** | 推理模型，对标 o1，开源！ |
| **DeepSeek Coder** | 代码专用，早期版本仍很好用 |

**为什么 DeepSeek 引爆全球？**
- DeepSeek V3 训练成本 **557 万美元**，而 GPT-4 约 1 亿美元
- 性能评测接近 GPT-4o，但价格便宜 **10-30 倍**
- **R1 完全开源**，可本地部署，不受出口限制
- 揭露了"算力不是唯一壁垒"，引发全球 AI 产业讨论

**API 价格对比（每百万 Token）：**
| | DeepSeek V3 | GPT-4o | Claude 3.5 Sonnet |
|-|-------------|--------|--------------------|
| 输入 | ¥1 | ~¥25 | ~¥22 |
| 输出 | ¥4 | ~¥100 | ~¥110 |

**入口：** chat.deepseek.com / platform.deepseek.com  
**本地部署：** ollama pull deepseek-r1

## 阿里云 · 通义千问（Qwen）

| 模型 | 特点 |
|------|------|
| **Qwen2.5-72B** | 综合能力强，开源旗舰 |
| **Qwen2.5-Coder** | 代码专用，对标 DeepSeek Coder |
| **Qwen-VL** | 多模态视觉理解 |
| **QwQ-32B** | 推理增强，对标 R1 |

**特点：**
- 阿里巴巴旗下，国内商业化最成熟之一
- **开源模型多，可私有化部署**
- 中文能力优秀
- 与阿里云生态深度整合（DingTalk、云服务）
- 国际化做得好，英文也不错

**入口：** tongyi.aliyun.com / dashscope.aliyuncs.com（API）

## 百度 · 文心一言（ERNIE）

| 模型 | 特点 |
|------|------|
| **ERNIE 4.0** | 百度旗舰，中文理解深 |
| **ERNIE Speed** | 快速低成本版本 |
| **文心一格** | 图像生成 |

**特点：**
- 国内第一批发布的 LLM，先发优势
- 中文语义理解好，搜索结合紧密
- 与百度搜索、地图、知识图谱结合
- 企业级应用（百度智能云·千帆）生态完善
- 代码能力相对弱，不是首选编程助手

**入口：** yiyan.baidu.com / cloud.baidu.com（API）

## 月之暗面 · Kimi

| 模型 | 特点 |
|------|------|
| **Kimi** | 超长上下文（200 万 Token！） |
| **Kimi k1.5** | 推理增强版 |

**特点：**
- **长文本处理能力国内最强**
- 可以上传整本书、长 PDF 进行分析
- 对话体验流畅，中文对话体验好
- 有联网搜索功能
- 适合：读长报告、分析合同、整理大量文档

**独特价值：** 如果你需要处理长文档，Kimi 是首选

**入口：** kimi.ai

## 合规与数据安全提示

> ⚠️ 使用国内 LLM API 时注意：

- 大多数国内模型**数据存在国内服务器**，有利于数据合规
- 部分企业级客户可申请**私有化部署**（避免数据上云）
- 涉及敏感业务数据时，优先考虑**本地部署开源模型**（DeepSeek R1 / Qwen）
- 了解各平台的数据使用协议
