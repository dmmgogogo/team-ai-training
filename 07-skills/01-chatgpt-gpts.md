# ⭐️ M7-01 · ChatGPT GPT Store 精选（开发向）

> GPT Store 里已有数百万个 GPT，这里整理开发者最常用、评分最高的精选列表。

---

## 如何找到并使用 GPT

1. 打开 ChatGPT → 左侧「探索 GPT」
2. 搜索名称 / 分类浏览
3. 点击「开始聊天」即可使用，无需额外配置

---

## 🔧 编程开发类

| GPT 名称 | 核心功能 | 适合场景 |
|---------|---------|---------|
| **Grimoire** | 代码向导，支持 20+ 语言，自动生成完整项目 | 从零开始建项目 |
| **Code Copilot** | 代码补全、重构、debug 全能助手 | 日常编码辅助 |
| **Diagrams: Show Me** | 代码/架构自动生成流程图、ER图、时序图 | 架构设计可视化 |
| **API Docs** | 读懂任意 API 文档，自动生成调用示例 | 接入第三方 API |
| **SQL Expert** | 自然语言转 SQL，复杂查询优化 | 数据库查询 |
| **RegEx Generator** | 用中文描述需求，生成正则表达式 | 文本处理 |

## 🐛 Debug & 代码审查类

| GPT 名称 | 核心功能 | 适合场景 |
|---------|---------|---------|
| **Bug Finder** | 专注找代码中的 bug，给出修复方案 | Code Review |

|---------|---------|---------|
| **System Design GPT** | 给需求，给出完整系统设计方案 | 架构规划 |
| **Database Designer** | 根据业务需求设计数据库表结构 | 数据库设计 |
| **API Designer** | 生成 RESTful / GraphQL API 设计文档 | 接口设计 |

---

## 💡 使用技巧

**1. 组合使用**
```
先用 System Design GPT 设计架构
→ 再用 Database Designer 设计表结构
→ 再用 API Designer 设计接口
→ 最后用 Code Copilot 生成代码
```

**2. 粘贴代码时加前置说明**
```
# 我的技术栈：Go + Beego，MySQL 5.7
[粘贴代码]
请帮我 review，重点关注安全漏洞
```

**3. 保存常用 GPT 到「我的 GPT」**
- 右上角固定常用 GPT，避免每次重新搜索
