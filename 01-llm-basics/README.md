# M1 · AI 大模型基础

## 模块概述

本模块介绍大语言模型（LLM）的基本概念、Transformer 架构、训练流程（预训练、微调、RLHF）以及常用术语速查，为零基础或希望系统梳理的同学打下基础。

**适合人群**：所有人
