# M10-01 · 什么是 AI Cloud Agent

> 不在你的电脑上跑的 AI，却能帮你完成完整的编程任务

---

## 从一个场景说起

你有没有遇到过这种情况：

```
你：帮我把这个旧项目从 Python 2 迁移到 Python 3
Cursor：好的，我来帮你...（改了一半，对话窗口太长崩了）
```

或者：

```
你：帮我跑一遍所有单元测试，把失败的都修掉
AI：我没有权限执行命令...
```

**本地 AI 工具的两个核心限制：**
1. **上下文窗口有限** — 任务太复杂，记不住前面做了什么
2. **运行环境缺失** — 不能真正在你的机器上跑命令

---

## Cloud Agent 是什么？

**Cloud Agent = 在云端独立环境里运行的 AI 自主任务执行器**

```
你发指令
    ↓
云端起一个干净的虚拟环境（VM / 容器）
    ↓
Agent 在里面：读代码 → 规划步骤 → 写代码 → 跑命令 → 验证结果
    ↓
（你去睡觉 / 开会，它一直在跑）
    ↓
完成后通知你：PR 已创建 / 结果已推送
```

**关键特征：**

| 特征 | 说明 |
|------|------|
| **隔离** | 在独立沙箱里运行，不影响你本机环境 |
| **异步** | 你不需要等着，它自己跑 |
| **长任务** | 可以跑几分钟甚至几小时 |
| **真实执行** | 能真正执行命令、跑测试、装依赖 |
| **结果可审查** | 完成后给你 diff / PR，你来 review |

---

## 和本地 AI 工具的区别

```
本地 AI 工具（Cursor Chat / Copilot）
─────────────────────────────────────
你的电脑 ──── AI ──── 建议代码 ──── 你来执行
              ↑
         只是对话，没有执行权

Cloud Agent（Cursor Background / Devin）
─────────────────────────────────────────
你发指令 ──── 云端 VM ──── AI 自主执行 ──── 结果推给你
                  ↑
           真实环境，完整执行链
```

| 对比维度 | 本地 AI 工具 | Cloud Agent |
|----------|-------------|-------------|
| 执行环境 | 你的电脑 | 云端隔离 VM |
| 任务时长 | 受上下文限制 | 可以跑几小时 |
| 是否异步 | ❌ 需要你在场 | ✅ 后台自动跑 |
| 能跑命令吗 | 有限 / 需确认 | ✅ 完整 shell |
| 适合任务 | 快速编辑、问答 | 大型重构、迁移 |
| 结果形式 | 内联代码建议 | PR / 文件 / 报告 |

## 主流 Cloud Agent 产品

### Cursor Background Agent
- Cursor 内置，与 IDE 深度整合
- 在云端 VM 里独立运行，可访问你的仓库
- 完成后直接推送分支，你来 review

### Devin（Cognition AI）
- 第一个商业化的"AI 软件工程师"
- 完整的任务规划能力：从需求到代码到测试
- 定价较高，适合复杂工程任务

### GitHub Copilot Workspace
- GitHub 官方出品，深度集成 GitHub 仓库
- 从 Issue 描述自动生成实现方案
- 支持在 Codespace 里执行

### Claude Code（Anthropic）
- 命令行工具，在终端里直接跑
- 可以在本地或 CI 环境使用
- 开发者首选，精准、可控

### OpenAI Codex（ChatGPT Codex）
- ChatGPT 内置的云端编程 Agent
- 独立沙箱环境，支持长任务

---

## 适合用 Cloud Agent 的场景

✅ **适合：**
- 大型代码库迁移（Python 2→3、框架升级）
- 批量修复同类型 Bug
- 编写完整功能模块 + 单元测试
- 代码审查 + 自动重构
- 生成文档 / README

❌ **不适合：**
- 需要你实时参与决策的任务
- 涉及生产数据库的直接操作
- 创意性工作（产品方向、架构设计）
- 任务描述本身就不清晰的需求

---

## 小结

Cloud Agent 不是更好的"代码补全"，它是一个**能真正干活的 AI 同事**：

```
给它一张工单 → 它去做 → 你来 review
```

核心价值：**把你从低价值的执行工作里解放出来**
